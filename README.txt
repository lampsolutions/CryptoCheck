# CryptoCheck Api

CryptoCheck allows you to monitor the blockchain and mempool for specific addresses.
Adding/Removing address jobs is done via rest api and receiving notifications about an event is done via callback.

For more information please visit our german website [www.cryptopanel.de](https://www.cryptopanel.de).
